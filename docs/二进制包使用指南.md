# 二进制包下载与使用指南

## 概述

Chat Room 提供了预编译的二进制可执行文件，无需安装 Node.js 环境即可运行。

## 支持的平台

- ✅ **macOS** (x64)
- ✅ **Linux** (x64)
- ✅ **Windows** (x64)

## 下载方式

### 方式 1: 从 GitHub Releases 下载

访问 [Releases 页面](https://github.com/yourusername/chat-room/releases) 下载对应平台的二进制文件。

**文件命名规则：**
- `chat-room-macos-x64.tar.gz` - macOS
- `chat-room-linux-x64.tar.gz` - Linux
- `chat-room-win-x64.zip` - Windows

### 方式 2: 使用 curl/wget 下载

**macOS / Linux:**
```bash
# macOS
curl -L https://github.com/yourusername/chat-room/releases/latest/download/chat-room-macos-x64.tar.gz | tar xz
chmod +x chat-room-macos

# Linux
curl -L https://github.com/yourusername/chat-room/releases/latest/download/chat-room-linux-x64.tar.gz | tar xz
chmod +x chat-room-linux
```

**Windows (PowerShell):**
```powershell
Invoke-WebRequest -Uri "https://github.com/yourusername/chat-room/releases/latest/download/chat-room-win-x64.zip" -OutFile "chat-room-win.zip"
Expand-Archive -Path chat-room-win.zip -DestinationPath .
```

## 安装步骤

### macOS

```bash
# 1. 下载
curl -L https://github.com/yourusername/chat-room/releases/latest/download/chat-room-macos-x64.tar.gz | tar xz

# 2. 添加执行权限
chmod +x chat-room-macos

# 3. 移动到 PATH（可选）
sudo mv chat-room-macos /usr/local/bin/chat-room

# 4. 验证安装
chat-room --version
```

### Linux

```bash
# 1. 下载
wget https://github.com/yourusername/chat-room/releases/latest/download/chat-room-linux-x64.tar.gz
tar -xzf chat-room-linux-x64.tar.gz

# 2. 添加执行权限
chmod +x chat-room-linux

# 3. 移动到 PATH（可选）
sudo mv chat-room-linux /usr/local/bin/chat-room

# 4. 验证安装
chat-room --version
```

### Windows

```powershell
# 1. 下载并解压
Invoke-WebRequest -Uri "https://github.com/yourusername/chat-room/releases/latest/download/chat-room-win-x64.zip" -OutFile "chat-room.zip"
Expand-Archive -Path chat-room.zip -DestinationPath .

# 2. 添加到 PATH（可选）
# 将包含 chat-room-win.exe 的目录添加到系统 PATH

# 3. 验证安装
chat-room-win.exe --version
```

## 配置

### 1. 首次运行

第一次运行时，会自动创建配置文件：

```bash
./chat-room-macos
```

会提示：
```
Config file not found. Creating default config...
Config created: /Users/yourname/.chat-room/config.json

Please configure your ZooKeeper address and nickname before starting.
```

### 2. 编辑配置文件

配置文件位置：
- **macOS/Linux**: `~/.chat-room/config.json`
- **Windows**: `%USERPROFILE%\.chat-room\config.json`

```json
{
  "userId": "user-unique-id",
  "nickname": "YourNickname",
  "p2pPort": 9000,
  "zkAddress": ["127.0.0.1:2181"],
  "recentRooms": []
}
```

**必填项：**
- `nickname` - 你的昵称
- `zkAddress` - ZooKeeper 服务地址

## 使用

### 启动应用

```bash
# macOS / Linux
./chat-room-macos
# 或如果在 PATH 中
chat-room

# Windows
chat-room-win.exe
```

### 命令行参数

```bash
# 指定配置文件
./chat-room-macos --config /path/to/config.json

# 指定 ZooKeeper 地址（覆盖配置文件）
./chat-room-macos --zk 192.168.1.100:2181

# 指定昵称（覆盖配置文件）
./chat-room-macos --nickname "MyNickname"

# 查看帮助
./chat-room-macos --help

# 查看版本
./chat-room-macos --version
```

## 系统集成

### macOS - 创建开机启动

创建 `~/Library/LaunchAgents/com.chatroom.plist`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.chatroom</string>
    <key>ProgramArguments</key>
    <array>
        <string>/usr/local/bin/chat-room</string>
    </array>
    <key>RunAtLoad</key>
    <true/>
    <key>KeepAlive</key>
    <false/>
</dict>
</plist>
```

加载服务：
```bash
launchctl load ~/Library/LaunchAgents/com.chatroom.plist
```

### Linux - systemd 服务

创建 `/etc/systemd/system/chat-room.service`:

```ini
[Unit]
Description=Chat Room Application
After=network.target

[Service]
Type=simple
User=your-username
ExecStart=/usr/local/bin/chat-room
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动服务：
```bash
sudo systemctl daemon-reload
sudo systemctl enable chat-room
sudo systemctl start chat-room
```

### Windows - 计划任务

使用 Task Scheduler 创建开机启动任务，或创建快捷方式放到启动文件夹：

```
C:\Users\YourName\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\
```

## 升级

### 手动升级

```bash
# 1. 下载新版本
# 2. 备份配置
cp ~/.chat-room/config.json ~/.chat-room/config.json.bak

# 3. 替换二进制文件
# macOS
sudo cp chat-room-macos /usr/local/bin/chat-room

# Linux
sudo cp chat-room-linux /usr/local/bin/chat-room

# Windows
# 替换 chat-room-win.exe
```

### 自动升级脚本

创建 `upgrade.sh`:

```bash
#!/bin/bash
VERSION=$(curl -s https://api.github.com/repos/yourusername/chat-room/releases/latest | grep '"tag_name"' | sed -E 's/.*"([^"]+)".*/\1/')
echo "Latest version: $VERSION"

# Download
curl -L "https://github.com/yourusername/chat-room/releases/download/$VERSION/chat-room-macos-x64.tar.gz" | tar xz

# Replace
chmod +x chat-room-macos
sudo mv chat-room-macos /usr/local/bin/chat-room

echo "Upgraded to $VERSION"
```

## 卸载

### macOS / Linux

```bash
# 1. 停止服务（如果有）
sudo systemctl stop chat-room  # Linux
launchctl unload ~/Library/LaunchAgents/com.chatroom.plist  # macOS

# 2. 删除二进制文件
sudo rm /usr/local/bin/chat-room

# 3. 删除配置和数据
rm -rf ~/.chat-room

# 4. 删除服务文件
sudo rm /etc/systemd/system/chat-room.service  # Linux
rm ~/Library/LaunchAgents/com.chatroom.plist  # macOS
```

### Windows

```powershell
# 1. 删除程序文件
Remove-Item "C:\Path\To\chat-room-win.exe"

# 2. 删除配置
Remove-Item -Recurse "$env:USERPROFILE\.chat-room"

# 3. 删除计划任务（如果有）
Unregister-ScheduledTask -TaskName "ChatRoom" -Confirm:$false
```

## 故障排除

### 问题 1: 无法执行二进制文件

**错误**: `Permission denied`

**解决方案**:
```bash
chmod +x chat-room-macos  # macOS/Linux
```

### 问题 2: macOS 无法打开（安全限制）

**错误**: `"chat-room-macos" cannot be opened because it is from an unidentified developer`

**解决方案**:
```bash
# 方法 1: 允许运行
sudo xattr -cr chat-room-macos
./chat-room-macos

# 方法 2: 在系统偏好设置中允许
# 系统偏好设置 -> 安全性与隐私 -> 仍要打开
```

### 问题 3: Linux 缺少依赖库

**错误**: `error while loading shared libraries: libxxx.so`

**解决方案**:
```bash
# Ubuntu/Debian
sudo apt-get install -y libglib2.0-0

# CentOS/RHEL
sudo yum install -y glib2
```

### 问题 4: 配置文件找不到

**错误**: `Config file not found`

**解决方案**:
```bash
# 手动创建配置目录
mkdir -p ~/.chat-room

# 创建配置文件
cat > ~/.chat-room/config.json << EOF
{
  "userId": "user-$(date +%s)",
  "nickname": "YourName",
  "p2pPort": 9000,
  "zkAddress": ["127.0.0.1:2181"],
  "recentRooms": []
}
EOF
```

## 文件校验

为确保下载的文件完整性和安全性，建议校验 SHA256 哈希值。

### 计算哈希

**macOS/Linux:**
```bash
shasum -a 256 chat-room-macos
```

**Windows (PowerShell):**
```powershell
certutil -hashfile chat-room-win.exe SHA256
```

### 验证哈希

在 [Releases 页面](https://github.com/yourusername/chat-room/releases) 查看发布的 SHA256 校验和，与本地计算的值对比。

## 性能优化

### 二进制文件大小

当前二进制文件大小约 **50-60 MB**（包含 Node.js 运行时）。

### 减小文件体积

可以使用 UPX 压缩（可选）：

```bash
# 安装 UPX
brew install upx  # macOS
sudo apt install upx  # Linux

# 压缩二进制文件
upx --best --lzma chat-room-macos

# 压缩后大小约 15-20 MB
```
