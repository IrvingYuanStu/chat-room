# 快速安装指南

## 前置要求

### 1. Node.js 环境

**需要 Node.js v16 或更高版本**

检查版本：
```bash
node -v
npm -v
```

如果没有安装：
```bash
# macOS
brew install node

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# 或访问官网下载
# https://nodejs.org/
```

### 2. ZooKeeper 服务

**需要可访问的 ZooKeeper 服务（v3.6+）**

#### 选项 A: 使用本地 ZooKeeper

```bash
# macOS
brew install zookeeper
brew services start zookeeper

# Ubuntu/Debian
sudo apt-get install zookeeper
sudo systemctl start zookeeper

# 验证运行
echo ruok | nc 127.0.0.1 2181
# 应返回: imok
```

#### 选项 B: 使用 Docker 运行 ZooKeeper

```bash
docker run -d \
  --name zookeeper \
  -p 2181:2181 \
  -e ZOO_MY_ID=1 \
  zookeeper:3.8
```

#### 选项 C: 使用远程 ZooKeeper

如果团队/公司已有 ZooKeeper 服务，直接使用其地址即可。

## 安装方式

### 方式 1: 一键安装脚本（推荐）

```bash
curl -fsSL https://raw.githubusercontent.com/yourusername/chat-room/main/install.sh | bash
```

安装脚本会：
- ✅ 检查 Node.js 环境
- ✅ 克隆代码仓库
- ✅ 安装依赖并编译
- ✅ 生成配置文件
- ✅ 询问 ZooKeeper 地址

### 方式 2: 手动安装

```bash
# 1. 克隆仓库
git clone https://github.com/yourusername/chat-room.git
cd chat-room

# 2. 安装依赖
npm install

# 3. 编译项目
npm run build

# 4. 配置 ZooKeeper 地址
mkdir -p ~/.chat-room
cat > ~/.chat-room/config.json << EOF
{
  "userId": "user-$(date +%s)",
  "nickname": "YourNickname",
  "p2pPort": 9000,
  "zkAddress": ["127.0.0.1:2181"],
  "recentRooms": []
}
EOF
```

### 方式 3: Docker 部署

```bash
# 1. 克隆仓库
git clone https://github.com/yourusername/chat-room.git
cd chat-room

# 2. 创建环境变量文件
cat > .env << EOF
ZK_ADDRESS=127.0.0.1:2181
EOF

# 3. 启动容器
docker-compose up -d

# 4. 查看日志
docker-compose logs -f
```

## 配置说明

配置文件位置：`~/.chat-room/config.json`

```json
{
  "userId": "user-unique-id",        // 用户唯一ID，自动生成
  "nickname": "YourNickname",         // 显示昵称
  "p2pPort": 9000,                    // P2P 监听端口
  "zkAddress": ["127.0.0.1:2181"],   // ZooKeeper 地址列表
  "recentRooms": []                   // 最近访问的聊天室
}
```

### 配置项说明

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `userId` | 用户唯一标识 | 自动生成，无需修改 |
| `nickname` | 聊天室显示昵称 | 修改为你的昵称 |
| `p2pPort` | P2P 通信端口 | 9000-9999 之间 |
| `zkAddress` | ZooKeeper 地址 | `["host:port"]` |
| `recentRooms` | 最近访问的房间 | 自动维护 |

## 启动应用

### 直接运行

```bash
cd ~/chat-room
node dist/index.js
```

### 使用 npm

```bash
cd ~/chat-room
npm start
```

### 创建全局命令（可选）

```bash
cd ~/chat-room
sudo npm link -g

# 现在可以在任何地方运行
chat-room
```

### 后台运行

使用 nohup：
```bash
cd ~/chat-room
nohup node dist/index.js > chat-room.log 2>&1 &

# 查看日志
tail -f chat-room.log
```

使用 PM2：
```bash
npm install -g pm2
pm2 start dist/index.js --name chat-room
pm2 save
pm2 startup
```

## 使用 systemd 管理（Linux）

创建服务文件：

```bash
sudo tee /etc/systemd/system/chat-room.service > /dev/null << EOF
[Unit]
Description=Chat Room Application
After=network.target

[Service]
Type=simple
User=$USER
WorkingDirectory=$HOME/chat-room
ExecStart=/usr/bin/node $HOME/chat-room/dist/index.js
Restart=on-failure
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF
```

启动服务：

```bash
sudo systemctl daemon-reload
sudo systemctl enable chat-room
sudo systemctl start chat-room
sudo systemctl status chat-room
```

## 故障排除

### 问题 1: Node.js 版本过低

**错误信息**: `Error: Node.js version too old`

**解决方案**:
```bash
# 升级 Node.js
brew upgrade node  # macOS
# 或重新安装最新版
```

### 问题 2: ZooKeeper 连接失败

**错误信息**: `ZooKeeper connection error` 或 `NO_NODE`

**解决方案**:
```bash
# 检查 ZooKeeper 是否运行
ps aux | grep zookeeper

# 测试连接
echo ruok | nc 127.0.0.1 2181
# 应返回: imok

# 查看 ZooKeeper 日志
tail -f /opt/zookeeper/logs/zookeeper.log  # Linux
tail -f /usr/local/var/log/zookeeper/zookeeper.log  # macOS
```

### 问题 3: 端口被占用

**错误信息**: `Error: listen EADDRINUSE: address already in use`

**解决方案**:
```bash
# 修改配置文件中的端口
vim ~/.chat-room/config.json
# 将 "p2pPort": 9000 改为其他端口
```

### 问题 4: 编译失败

**错误信息**: TypeScript 编译错误

**解决方案**:
```bash
# 清理并重新安装
cd ~/chat-room
rm -rf node_modules dist
npm install
npm run build
```

## 更新

```bash
cd ~/chat-room
git pull origin main

```

## 卸载

```bash
# 停止服务
sudo systemctl stop chat-room  # 如果使用了 systemd
pm2 stop chat-room             # 如果使用了 PM2

# 删除文件
rm -rf ~/chat-room
rm -rf ~/.chat-room

# 删除全局命令
sudo npm unlink -g chat-room

# 删除服务
sudo rm /etc/systemd/system/chat-room.service
```
